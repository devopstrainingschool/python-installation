# python-installation

## Installing python
```
sudo yum install python36 -y
```
## check the version
```
python3 -V
```
## installing pip
```
sudo yum install epel-release
```
```
sudo yum update -y
```
```
yum install python-pip
```
## check the pip version
```
pip -V
```
## Upgrade pip
```
pip install --upgrade pip==18.1
```
## check again
```
pip -V
```



# some pip commands:
```
pip install -r requirements.txt
```
## to run an application
```
python2 app.py
```
